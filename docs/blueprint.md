# **App Name**: OpenSource Sage

## Core Features:

- Repository Discovery: Discover open-source repositories based on selected tech stack, experience level, and learning goals.
- Personalized Recommendations: Suggest relevant open-source repositories based on user's specified tech stack, goals, and experience level.
- AI-Powered Explanation: Use Gemini API to provide flowchart-style breakdown and explanation of selected repositories. LLM will use its tool to incorporate resources for code explanation.
- Interactive Flowchart: Render AI-generated explanation as an interactive flowchart using Mermaid.js.
- Learning Resource Integration: Integrate learning resources such as videos, documentation, and tutorials related to the repository's tech stack.
- Repository Detail View: In a dedicated view, display the name, the description, the flowchart, and a list of modules. Offer buttons to 'Star', 'Fork', and 'Save for later'.
- User Preference Collection: Onboarding flow with step-by-step guide to allow users to select technologies of interest, their learning goals, and their experience levels. Store data locally.

## Style Guidelines:

- Background color: Dark navy blue (#1A202C) to create a developer-friendly and focused environment.
- Primary color: Electric purple (#A78BFA) to bring some vivid color to the muted backdrop, and provide engaging visual accents.
- Accent color: Soft violet (#C084FC) to complement the primary color, creating depth and visual interest in components.
- Headline font: 'Space Grotesk', sans-serif. Body font: 'Inter', sans-serif. Space Grotesk gives a futuristic impression to headlines; Inter will ensure readability in body copy.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use simple, clear icons from 'lucide-react' to represent tech stacks and learning goals.
- Follow a clean, card-based layout with clear separation of content sections, inspired by Vercel and Readme. Dark-first theme. Mobile-first responsiveness.
